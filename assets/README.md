# Assets Directory

Папка для хранения графических ресурсов приложения UART Control.

## Структура

```
assets/
├── icons/          # Иконки приложения и компонентов
│   ├── logo.png    # Логотип для splash экрана (140x140 px)
│   ├── app.png     # Иконка приложения (256x256 px)
│   └── ...
├── images/         # Фотографии и общие изображения
│   ├── splash.png  # Изображение для splash экрана (400x500 px)
│   └── ...
└── README.md
```

## Рекомендации

### Для Splash Экрана
- **Логотип** (`icons/logo.png`): 140×140 пикселей, PNG с альфа-каналом
- **Фоновое изображение** (`images/splash.png`): 400×500 пикселей, PNG с прозрачностью

### Форматы
- Предпочтительные форматы: **PNG**, **SVG**
- Все иконки должны иметь прозрачный фон (альфа-канал)
- Поддерживаемые форматы: PNG, JPG, BMP, GIF

### Размеры
- Логотипы: 128×128 px или 256×256 px
- Иконки кнопок: 16×16 px, 24×24 px, 32×32 px
- Splash экран: 400×500 px
- Фоны: 1920×1080 px или выше

## Использование в коде

```python
from PySide6.QtGui import QPixmap

# Загрузка логотипа
logo = QPixmap("assets/icons/logo.png")

# Загрузка фонового изображения
image = QPixmap("assets/images/splash.png")
```

## Совместимость с темами

При создании изображений:
- Для **тёмной темы**: используйте светлые цвета (белый, серебристый)
- Для **светлой темы**: используйте тёмные цвета (чёрный, серый)
- Для **универсальных**: используйте нейтральные цвета (синий, зелёный, голубой)

## Текущие ресурсы

Приложение автоматически использует:
- `assets/icons/logo.png` для splash экрана (если файл существует)
- Если файл не найден, показывается текстовый логотип "UART"
